## 🚀 Getting Started (Frontend)

1. **Clone repository**
  ```bash
  git clone <repository-url>
  ```
2. **Navigate to the frontend folder:**
  ```bash
  cd frontend
  ```
3. **Install dependencies:**
  ```bash
  npm install
  ```
4. **Start the development server:**
  ```bash
  npm run dev
  ```
5. **Build for production:**
  ```bash
  npm run build
  ```
6. **Preview the production build:**
  ```bash
  npm run preview
  ```
